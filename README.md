### Applied Research

- **Real-time computer-vision automation** — low-latency detection→action pipelines, state-machine behavior orchestration, input-event bridging for desktop automation frameworks
- **Windows runtime & binary analysis** — IL2CPP metadata parsing, embedded scripting-engine inspection, entropy profiling of encrypted configuration artifacts
- **Device identity & integrity research** — hardware / OS fingerprint enumeration, registry-based machine-ID schemes, sandboxing and syscall-hook effects on process isolation
- **Production LLM agent systems** — multi-step tool-use workflows, retrieval pipelines, context & memory management, orchestration layers for automation workloads

Private repositories available on request.

### Engineering Principles

- Runnable output over polished descriptions
- Systems-analysis techniques applied to computer-vision pipelines; LLM agents as the orchestration layer
- Operator-grade feedback — readiness indicators, live logs, artifact previews — lands before accuracy optimization
- Every label, weight, dataset, and run carries the signature that produced it
- Prototypes are scoped so they can become commercial tools without a rewrite
