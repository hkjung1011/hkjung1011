### 🧭 Principles I care about

- **Recovery-first over model-first** — bad data is invisible until you give the operator a readiness %
- **Provenance everywhere** — every auto-label carries the model signature that produced it
- **Group-level splits** — never average scores across incompatible datasets or weight versions
- **Offline-capable desktop delivery** — factory PCs don't get internet
