- **Windows runtime & binary analysis** — IL2CPP metadata parsing, embedded scripting-engine inspection, entropy profiling of encrypted configuration artifacts

### 🧭 Engineering Principles

- Runnable output over polished descriptions
- Systems-analysis techniques applied to computer-vision pipelines; LLM agents as the orchestration layer
- Operator-grade feedback — readiness indicators, live logs, artifact previews — lands before accuracy optimization
- Every label, weight, dataset, and run carries the signature that produced it
- Prototypes are scoped so they can become commercial tools without a rewrite
